## Project Description
This project demonstrates the implementation of **Unit Testing** and **Integration Testing** using the **pytest** framework in Python.  
A simple banking application is developed with core functions such as deposit, withdraw, interest calculation, loan eligibility, and fund transfer. Unit tests verify individual functions, while integration tests validate the interaction between multiple functions.

- Pytest
- Pytest-HTML
